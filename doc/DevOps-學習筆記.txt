2. RBAC

sys-admin: 特殊帳號
- 群組成員：SA001
- 系統權限：System Admin
- 專案權限：null
- 倉庫權限：null

sys-manager: 管制科
- 群組成員：SM001
- 系統權限：Admin
- 專案權限：null
- 倉庫權限：null

dev-manager: AP科長
- 群組成員：DM001
- 系統權限：Bitbucket User
- 專案權限：Create repository
- 倉庫權限：Write

dev-user: AP科員
- 群組成員：DU001、DU002
- 系統權限：Bitbucket User
- 專案權限：Read
- 倉庫權限：Write
